Requirements

All Hack Clubbers globally under the age of 18 are eligible! Any US high school students over the age of 18 can join.

Your website should:

  • Be original (No copy pasting)

  • Have 2 Html pages with content

  • Have something fun & unique!

  • Have 5+ HTML and CSS features for each page

  • Have 3 Javascript Functions for each page

  • Be made recently

  • Be static (uses HTML + CSS and JavaScript)

  • Have text, color and a image
